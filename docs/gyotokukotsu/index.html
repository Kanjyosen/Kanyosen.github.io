サークル概要

このサークルでは千葉県市川市行徳を拠点に、電子工作やバス沿線情報などを発信してまいります。
名前の「行徳交通」は、交通系の企画を行うということだけでなく、このサークルの活動でこの締め切られた時代に縁をひらく、一つの交通手段として役立ちたいという思いも込めています。
