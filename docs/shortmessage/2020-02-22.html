今日なんですが、しばらくデビットカードを使ってみた感想をば。平凡ですがなにとぞ。
特に加盟店関係では困りませんでした。VISAのネットワークが使えるので大抵のお店で使える。
ただ、なんていうか、クレジットカードとは違うのにクレジットカードと言えと言われるのはなんかね...
実際何回かクレジットカードと言おうとしてもなんか言えずに結局デビットカードと言ってしまいましたね...
最終的にたどり着いた境地が、「クレジット扱いで」。
結局之が早いしデビットカードということも伝わるし。伝わる必要ないけど。
まあ、前回紹介した市川市本八幡発祥の伝説的イタリアンリストランテでは使えませんが...
［PS］
秋葉原のアトレで切れたりしたり、カード店舗拡大もされるそうです。完全にしつれいしました。
［PS終わり］
更に、使いすぎなかったりっていうのもいい。まあ口座残額に衝突したこと無いけど。
今使っているのはSMBCデビットです。行徳駅前に三井住友銀行の有人支店があったので、そこで口座を作るついでに申し込みました。
ちなみに、カードのデザインは４種類から選べます。私はあえてデフォルトにしました。
あの緑色と白の配置好きなんですよね。銀色もちゃんと配していてすばらしい。
カードは、ICと磁気ストライプを搭載しています。
一応VISA touchを搭載しているんですが、実在するの？ってくらい使ってみた試しがないので今度マクドネル・ダグラスで使ってみます。
iD debitも搭載してるらしいんですが、普通のデビットで事足りるので使ってないです。なんかﾓｯﾀｲﾅｲ。
