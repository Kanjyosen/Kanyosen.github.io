マイフェス2019お疲れ様でした
まず最初に、マイフェス2019お疲れ様でしたー！ 今回、都市博の人として参加させていただきました。PiOすっごくでかいですね。特に台展示ホール前のアトリウム。 天井も高い、鉄骨が組まれてる、ライティングばっちり。こんなに夜におしゃれな場所はない。 さらに、大展示ホールの西側にある風防室。あそこはめっちゃいい音取れますよ。 空き時間でこんなもの作って見ました。 ...そういうことはさておき、今回やった仕事が以下の通りでして。
  1. 都市博のポスター設営
  2. 西本部の留守番とか落し物対応とか
  3. 会場を眺める
  4. 都市博と会場外周の落し物巡検
この通り。今見てみると結構やったなぁ、とは思ったりします。 会場はうちとしてはメガトン級に広く、私も最初は「ただの候補だと言ってくれ」とか言ってました笑 （今となってはここでやったほうがむしろ良かったのかもしれませんが） 事前リサーチによると、「声月」なるボイスロイドのイベントが開催されたりと、界隈では有名な会場のようです。 さて、そろそろ話をマイフェスの方に戻しましょう。 たくさんの方々が来てくださって非常に盛り上がってましたね。 そして、多彩なコーナーたち。VR、同人即売会、人狼… こんなん盛り上がらないはずがない。すごい。 そして、今回マイフェスの会場を眺めてて思ったのが、子連れさんが多いってことですね。 子供さんがクラフターになって、マイクラ界隈が面白いアイデアでまたいい感じに栄えられたら、私の本望です。 最後になりますが、都市博の企画の段階から褒めたり引っ張ったりしてくれたなああよさん（a.k.a.主犯、酒販、中込 椋）とにぐとめあれさん、そして出展関係の助言をたくさんくれた染宮さん、本当にありがとうございました。 そして、マイフェスの運営に関わってくださった全ての方々、お疲れ様でした！ありがとう！
