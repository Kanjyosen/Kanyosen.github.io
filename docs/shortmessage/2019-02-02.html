こんちは。
さて、今日は#KittenLT1に出演してきました！
A枠の2番目ですので興味ある方は是非どうぞ。
さて、今回は会場からお送りします。
スタッフさんがなかなかのプロな上に、ここで簡単なヒラサワライブでもできるかもってくらいの機材。
TeamKittenさん、イベント部発足してもいいんじゃないんですかね...
だって、アプリの出来もいいし。App Storeで数年後に「TeamKitten」っていうアプリが配信されてそう。
あのデザインと機能だとAppleの審査通るんじゃないですかね。
会場はこじんまりとしているけどなかなかいい会場です。
実績積んで、私もこういう会場で「動くパフォーマンス」についてのイベントやりてぇ！めっちゃ！
とりあえず実績積んでから強い人を後ろ盾にしてやろうかな。
といった感じで、ちょっと早いけど今回はここまでです。ご覧いただきありがとうございました！
